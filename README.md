## 🎨 Design & Technologies

- **HTML5 / CSS3 / JavaScript Vanilla** — Aucun framework requis
- **Google Fonts** — Bebas Neue + Barlow (chargés via CDN)
- **Responsive Design** — Mobile, tablette et desktop
- **Animations CSS** — Scroll reveal, float, pulse, fadeUp
- **Intersection Observer API** — Animations au défilement
- **Slider automatique** — Défilement des coloris toutes les 3 secondes
